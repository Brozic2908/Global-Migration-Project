# Hướng dẫn chạy code Đồ án Di cư

## 1. Cài đặt thư viện
Mở terminal/command prompt và chạy lệnh:
`pip install -r requirements.txt`

## 2. Cách chạy
- Mở thư mục `notebooks/` bằng Jupyter Notebook hoặc VS Code.
- Chạy lần lượt từ file `01_...` đến `04_...`.
- Kết quả bảng biểu và hình ảnh sẽ tự động xuất ra thư mục `output/`.
